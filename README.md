# Time & Scheduler Plugin

Botに「定期実行」や「予約投稿」などの時間制御機能を追加するCDMプラグインです。
このプラグインは、特定の時間や間隔で処理を開始する「イベントブロック」を提供します。

## 提供されるブロック

### 1. 定期実行 (Cron Event)
指定した間隔（毎日、毎時、○分おきなど）で、中に配置されたブロックを繰り返し実行します。
- **タイプ**: イベント開始ブロック（ハットブロック）
- **設定項目**: 毎日 / 毎時 / 1分おき / 5分おき / 30分おき

### 2. 指定時間 (At Time Event)
指定した時刻（例: 12:00）になった時に、中に配置されたブロックを実行します。
- **タイプ**: イベント開始ブロック（ハットブロック）
- **設定項目**: 時刻（HH:mm形式）

### 3. 待機 (Wait)
処理を一時停止させます。他のイベントブロックやコマンドブロックの中で使用します。
- **タイプ**: 文ブロック（ステートメント）
- **設定項目**: 数値（秒 / 分）

## インストール方法
1. `scheduler-plugin.zip` をダウンロードします。
2. CDMのプラグイン管理画面で「+ ZIPからインストール」を選択し、ZIPをアップロードします。
3. ツールボックスの「時間・スケジュール」カテゴリーからブロックを使用できます。
